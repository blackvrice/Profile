프로그래밍 언어 마다 변수 선언 방법 사용하는 자료형이 다르다.
허나 크게적으로 정수만 사용하는 정수형 자료형, 실수도 포함하는 실수형 자료형, 참과 거짓만 있는 논리형 자료형이 있다.
정수형 자료형이란 ···, -2, -1, 0, 1, 2, ··· 형식이고
실수형 자료형이란 ···, -0.2, ···, -0.1, ···, 0, ···, 0.1, ···, 0.2, ··· 형식이다.
또한 컴퓨터의 메모리크기가 무한이 아니기 때문에 자료형 마다 정해진 메모리 크기가 있다

C, C++

C언어에서 기본적으로 제공하는 자료형이 8개 있다.
정수형 자료형 : char, short, int, long, long long
실수형 자료형 : float, double
논리형 자료형 : bool

자료형 메모리 크기 값
bool 1bit false(0),true(1)
char 1byte -128 ~ 127
short 2byte -32,768 ~ 32,767
int 4byte -2,147,483,648 ~ 2,147,483,647
long long 8byte -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
float 4byte 3.4E+/-38(7개의 자릿수)
double 8byte 1.7E+/-308(15개의 자릿수)
